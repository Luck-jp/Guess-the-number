🔢 Guess the Number Game (Python)

A command-line number guessing game where the player tries to guess a randomly generated number within a limited number of attempts.

🚀 Features

Random number generation (1–100)

Two difficulty levels (easy & hard)

Limited attempts system

Hints (Too high / Too low)

Replay option

🛠️ Tech Used

Python 3

▶️ How to Run

Clone the repository:
git clone https://github.com/Luck-jp/guess-the-number-python.git

Navigate to the folder:
cd guess-the-number-python

Run the program:
python main.py

📌 Project Purpose

This project was built to understand:

Conditional logic

Loops and control flow

Functions and modular design

Random number generation
